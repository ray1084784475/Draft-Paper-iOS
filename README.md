## 简介

### 主要特性

🎨 **丰富的绘图工具**
- 多种画笔选择：钢笔、铅笔、马克笔、橡皮擦
- 8种预设颜色 + 完整自定义颜色选择器
- 1-50px 可调节画笔粗细，实时预览效果



🎯 **直观的用户界面**
- 现代化 SwiftUI 设计，简洁美观
- 工具栏智能布局，操作便捷
- 实时视觉反馈，交互流畅

## 系统要求

- **设备**：iPhone/iPad (iPad最佳，支持 Apple Pencil 的iPad型号更佳)
- **系统**：iPadOS 14.0 或更高版本
- **开发**：Xcode 12.0+, Swift 5.3+

## 安装使用

### 开发环境运行

1. 克隆项目到本地：
```bash
git clone https://github.com/ray1084784475/draft-paper-ios.git
```

2. 使用 Xcode 打开项目：
```bash
cd draft-paper-ios
open 草稿纸.xcodeproj
```

3. 选择目标设备为 iPhone/iPad
4. 编译并运行应用 (⌘ + R)

### 在 Swift Playgrounds 中运行

1. 将所有 Swift 文件复制到 Playgrounds 项目
2. 确保导入必要的框架
3. 在 iPad 版 Swift Playgrounds 中运行

## 使用指南

### 基础绘图

-**选择颜色**：点击颜色圆圈选择预设颜色，或点击调色板图标选择自定义颜色

-**选择工具**：在工具栏中选择钢笔、铅笔、马克笔或橡皮擦

-**调节粗细**：使用滑动条调整画笔大小，实时预览效果

-**开始绘制**：在画布上自由绘制，享受无边界创作

-**清除画布**：点击垃圾桶图标一键清空所有内容
