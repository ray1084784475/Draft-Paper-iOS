## User Guide

### Basic Drawing

- **Select Color**: Click color circles to choose preset colors, or click the palette icon for custom color selection
- **Select Tool**: Choose from Pen, Pencil, Marker, or Eraser in the toolbar
- **Adjust Thickness**: Use the slider to adjust brush size with real-time preview
- **Start Drawing**: Draw freely on the canvas, enjoy unlimited creation
- **Clear Canvas**: Click the trash icon to clear all content with one tap
